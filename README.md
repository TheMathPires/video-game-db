## Getting Started

### Prerequisites

- node
- npm
- yarn

### Running

1. Clone the repo

```sh
git clone https://github.com/TheMathPires/video-game-db.git
```

2. Install NPM packages

```sh
npm install
```

3. Run development mode

```sh
npm start
```
